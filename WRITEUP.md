# Write-up Template

### Analyze, choose, and justify the appropriate resource option for deploying the app.

I chose Azure App Service because it’s cost-effective, scales automatically, and has built-in high availability. It’s also simpler to manage and doesn’t require as much coding skill. This makes it a practical choice for my CMS app, ensuring smooth scalability and an easy workflow.

### Assess app changes that would change your decision.

If my CMS app needed custom OS configurations, had to run legacy software, or required more control over the infrastructure, I would choose Azure VMs. Additionally, if I expected very high, unpredictable traffic that needed manual scaling, VMs would offer the necessary flexibility. These changes would make VMs a better fit despite the higher management overhead.
